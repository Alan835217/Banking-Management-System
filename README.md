# Banking-Management-System
Banking Management System (Python + SQLite)

A simple Banking Management System built using Python and SQLite.

This project allows users to create accounts, securely log in, and perform basic banking operations such as deposit, withdrawal, and balance checking also sending money to the another account number.

Features

Create new bank account
Secure PIN-based login (hashed PIN storage)
Deposit money
Withdraw money
Check account balance
Transaction safety with database integration
Admin login to see users and their amount

Technologies Used

Python 3
SQLite3 (built-in database)
Datetime module
Hashlib (for PIN encryption)

How to Run

open the app.py file
run streamlit run app.py to run the bank management project

Anyone can use the code if needed
